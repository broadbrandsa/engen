## What Is Excluded

- **User authentication / login:** Not in scope
- **CMS integration:** No Contentful, Sanity, or similar — content is file-based
- **E-commerce / payments:** Not in scope
- **Backend APIs / databases:** Not in scope at launch
- **Multilingual / i18n:** Not in scope
- **Accessibility audit:** Basic a11y practices applied; formal audit not included
- **Mobile app:** Web only
