# Project Overview

## What This Site Is

This is a standalone proposal website built for Engen. It presents a structured, professional pitch to a target client or stakeholder audience. The site is designed to communicate strategy, scope, deliverables, and value clearly — without requiring backend systems or internal data access.

## Purpose

- Present a compelling, well-designed proposal in a web format
- Replace or supplement static PDF proposals with an interactive, navigable experience
- Serve as a shareable link for stakeholders to review at their own pace

## Target Audience

- Decision-makers and executives at the prospective client organization
- Internal stakeholders reviewing and approving the proposal
- Any party with access to the shared proposal link

## Deployment Environment

- **Platform:** Vercel
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS v4 + shadcn UI
- **Language:** TypeScript
- **Package Manager:** pnpm
